# GPA Calculator
This application is designed to predict CGPA without the grades of current courses.

## How it works
CGPA is calculated based on
- User's input
- Assignment percentage
- Course percentage
 
## How to run

Clone this repo to local and use command:
    $`bundle exec rails server`
